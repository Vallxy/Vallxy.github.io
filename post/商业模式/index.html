商业模式

《商业模式新生代读书笔记》

[toc]

1. 定义

一个商业模式描述的的是一个组织创造、传递以及获得价值的基本原理

2.九大模块

2.1 客户细分 customer segments

描述了一家企业想要获得的和期望服务的不同的目标人群和机构

2.2 价值主张 value propositions

描述的是为某一客户群体提供能为其创造价值的产品和服务

是客户选择你而不是其他人的原因

2.3 渠道道路 channels

描述的是一家企业如何同它的客户群体达成沟通并建立联系，以向对方传递自身的价值主张

2.4 客户关系 customers relationships

描述的是一家企业针对某一个客户群体所建立的客户关系的类型

2.5 收入来源 revenue streams

代表了企业从每一个客户群体获得的现金收益（须从收益中扣除成本得到利润）

2.6 核心资源 key resources

描述的是保证一个商业模式顺利运行所需的最重要的资产

2.7 关键业务 key activities

描述的是保障其商业模式正常运行所需要的最重要的事

2.8 重要合作 key partnerships

描述的是保证一个商业模式顺利运行所需的供应商和合作伙伴网络

2.9 成本结构 cost structure

描述的是运营一个商业模式所发生的全部成本

3. 商业画布

在画布上贴上项目相关的东西

4. 商业模式类型

4.1 分拆商业模式

这一概念将企业从事的活动分为三种不同类型：客户关系管理、新产品开发以及基础设施管理。每种类型的活动有着不同的经济、竞争和文化规则。这三种类型可能共存于同一家企业中，但理想情况下，它们各自存在于相互独立的实体中以避免冲突或不必要的增长。

例如：电信公司

拆分为三部分

设备生产商、减负后的运营商、内容提供商

4.2 长尾商业模式

长尾商业模式在于少量多种地销售自己的产品：它致力于提供相当多种类的小众产品，而其中的每一种卖出量相对很少。将这些小众产品的销售汇总，所得收入可以像传统销售模式一样可观，它不同于传统模式，已销售少数的明星产品负担起绝大部分收益。长尾商业模式要求低库存成本以及强大的平台以保证小众商品能够及时被感兴趣的买家获得。

例如：图书出版行业的转型

旧的模式：作者写稿子，出版社审阅，择优发行 (出版营销和批发收益)

新的模式：通过在线商城，向作家提供排版、印刷工具，帮助他们推广，书只在销售的时候被印刷。

​ (平台管理与开发+销售佣金+出版服务费)

4.3 多边平台商业模式

多边平台将两个或更多独立但相互依存的客户群体连接在一起。这样的平台对于平台中某一群体的价值在于平台中其他客户群体的存在。平台通过促进不同群体间的互动而创造价值。一个多边平台的价值在于它所吸引的用户数量的增加，这种现象被称为网络效应。

谷歌从广告商这一客户群体赚钱，对于上网浏览者和内容提供者给予补贴

苹果 app store

4.4 免费的商业模式

在免费商业模式中，至少有一个关键的客户群体是可以持续免费地享受服务的。新的模式使得免费提供服务成为可能。不付费的客户所获得的财务支持来自商业模式中另一个客户群体。

1.基于多变平台的免费商品

2.免费的基本服务，可选的增值服务

3.钓鱼模式，以一个免费或者很便宜的初始价格吸引客户，并引诱客户使其进入重复购买的状态

4.5 开放的商业模式

开放的商业模式适用于通过与外部合作伙伴系统地配合而创造和获取价值的企业。这种模式可以是“由外而内”地与企业内部尝试来自外部的理念，或者“由内而外”地向外部合作伙伴输出公司无用的理念或资产。

外部技术可以引入从而盈利

内部技术可以导出从而变现

例如：宝洁公司在自身商业模式中搭起三座桥梁：高科技企业家、互联网平台、退休专家

5. 商业模式设计

5.1 客户洞察 customer insights

客户视角是商业模式设计的指导性原则，客户的观点决定了我们选择怎样的价值主张、渠道、客户关系和收益来源。

关键是通过用户的眼睛来看商业模式；关注哪些用户，忽略那些用户

5.2 构思 ideation

分为生成与整合两个阶段

忽略现状、忘记过去、不要盯着竞争对手、挑战正统思维

按照创新的焦点分类：资源驱动、供给驱动、客户驱动、财务驱动

5.3 视觉化思考 visual thinking

通过便利贴来实现视觉化

5.4 模型构建 prototyping

通过制作商业模式模型，能够帮助探索创意

构建不同规模的模型

5.5 讲故事 storytelling

新的或者创新性的商业模式可能是难以描述或者难以理解的，讲故事能够更好地沟通，做好内在逻辑准备

公司视角和客户视角

5.6 场景 scenarios

场景可以描述不同的客户结构

场景可以描述一种商业模式未来可能的竞争环境

6. 战略

6.1 商业模式环境

背景、设计动力、限制

商业模式是在特定的环境下被设计和执行的。

环境的四个主要领域：1.市场影响力 2.行业影响力 3.关键趋势 4.宏观经济影响

6.2 评估商业模式

定期评估商业模式是一项重要的管理活动。这能让一个组织评估它的市场地位的健康程度，并且做出相应的调整。

从总体角度评估；评估每个模块

SWOT评估

6.3 从商业模式的视角看蓝海战略

四项行动架构

  1. 行业中哪些看起来理所当然的要素应该被删除
  2. 哪些要素应该被大幅提升到行业标准之上
  3. 哪些要素应该大幅削减到行业标准之下
  4. 哪些行业中从未提供的要素是应该被创造出来的

6.4 管理多种商业模式

子模块整合还是剥离

7. 商业模式设计流程

动员、理解、设计、实施、管理
